# Roadmap

## Phase 1: Foundation

- Monorepo structure
- Backend HTTP server
- MySQL connection and transaction helper
- Standard JSON response and error format
- CORS, logger, and recovery middleware
- Migration and seed folders
- Vue app shell with Tailwind, Pinia, and Vue Router
- Responsive app layout with sidebar, drawer, topbar, alert badge, and user menu placeholder
- Placeholder pages for all required modules
- Reusable UI component set

## Phase 2: Authentication and Roles

- Login API
- Password hashing
- Session or JWT handling
- Route guards
- ADMIN, MANAGER, CASHIER permissions

## Phase 3: Inventory and Products

- Categories
- Products
- Locations
- Multi-location product stock
- Restock, adjustment, movement history, and transfer flows

## Phase 4: POS and Sales

- POS cart
- Barcode input
- Sale transaction
- Receipt
- Sales history and cancellation

## Phase 5: Reporting and Operations

- Alerts
- Reports
- Import/export
- Suppliers
- Purchase orders
- Settings
- LINE notifications
